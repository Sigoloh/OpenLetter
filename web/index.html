Oppen Letter

Newsletter by everyone, to everyone

  • What is Open Letter?

    • Open Letter is my first project made for the Open Source community. Its objective is to democratize the creation of email lists, sending news and analyzing analytics for all those interested in content production. To achive such goal, I've been using Typescript and Express to do the backend and (by now) html-5, css-3 and vanilla javascript to do the frontend

  • What do we have now?

    • So, at this momment, Open Letter is working and doing what it is meant to do. Functionalities as:
    • Content producer account management (create profile, send email, etc)
    • Creation of subscribers list
    • Email delivery

  • Whate will be added soon?

    • This list represents features that MUST be added before the actual release of Open Letter
    • Enhance security
    • Create the analytics tool
    • Create the email editor built in the frontend
    • Transform the frontend in VueJS, ReactJS or any other frontend javascript framework

  • Whate will be added not that soon?

    • This list represents features that I want to add in Open Letter but are not as important as security, analytics, etc
    • Something like an Open Letter own Markdown in order to build bether email bodies
    • The possibility to build your own subscriber page

Want to contribute?

Well, I'm currently not accepting monetary help now, but it would be cool if someone could help me with hosting. However, if you want to contribute with code, you can go to the Open Letter github repository, create a fork and make your own features, and, by the end, create a pull request.
Another greate way to contribute is to share the project with your friends, family or any other person who you think may be interested in something like Open Letter.
